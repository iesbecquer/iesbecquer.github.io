Etiqueta párrafo: <p>···</p>

Sirve para crear un párrafo de texto.

Ejemplo:

<p>Esto es un primer párrafo.</p>
    <p>Esto es un segundo párrafo.</p>

Tiene como salida:

Esto es un primer párrafo.

Esto es un segundo párrafo.

Etiqueta salto de línea: <br>

Sirve para forzar que termine una línea y comience desde la izquierda en otro renglón.

Esta etiqueta no hay que cerrarla.

Ejemplo:

<p>Esto es una primera línea.<br>
    Esto es una segunda línea.</p>

Tiene como salida:

Esto es un primer párrafo.
Esto es un segundo párrafo.

Etiqueta para introducir imágenes: <img src="···">···</img>

Introduce una imagen en la página web.

Tiene una serie de atributos que hay que conocer, como son:

  • src: Dirección donde se encuentra la imagen. Es obligatorio ponerlo.
  • width: ancho de la imagen. Es opcional.
  • height: alto de la imagen. Es opcional.
  • alt: descripción de la imagen. Es opcional.

Esta etiqueta no hay que cerrarla.

Ejemplo:

<img src="https://www.google.es/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png">

Etiqueta para introducir enlaces: <a href"···">···</a>

Introduce un enlace en la página web.

Tiene una serie de atributos que hay que conocer, como son:

  • href: Dirección web que se carga al hacer clic en el enlace.

Ejemplo:

<a href="https://www.google.com">Enlace a Google.</a>

Tiene como salida:

Enlace a Google
